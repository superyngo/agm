AGM TUI Layout Mockups

Interactive mockups for the AGM project redesign — Source & Tool TUI views

Source TUI — Normal Mode
The tree view lists skill repos and their individual skills. The status bar at the bottom shows a yellow progress indicator while a background update is running. Press l to open the log overlay.

Source TUI — Info Popup

Tool TUI — Main View
Displays the central AGM configuration and all registered tools in a tree. Each tool shows its link status for prompt, skills, agents, plus config files (settings, auth, mcp). Tools are collapsible. The highlighted cursor line (yellow on dark gray) indicates the currently selected row. ✗ not installed tools are collapsed and dimmed.

Tool TUI — File Picker Popup
When pressing e on a row that maps to multiple files (e.g. a tool's settings or mcp config with multiple paths), a small centered popup lets you choose which file to open. Use ↑↓ to navigate, ⏎ to open, or Esc to cancel.

Tool TUI — Inline Path Editor
When pressing e on a central path row (skills, agents, prompt, source), an inline text-editing popup appears pre-filled with the current path. Edit the path and press ⏎ to save, or Esc to cancel. The blinking cursor (█) shows the edit position.
